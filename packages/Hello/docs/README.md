### MeshPhongMaterial 属性

|    参数     |                                                                      说明                                                                       |
| :---------: | :---------------------------------------------------------------------------------------------------------------------------------------------: |
|   `color`   |                                                             材料的颜色，默认为白色                                                              |
| `emissive`  |                                                材质本身发出的颜色，不受其它光照影响，默认为黑色                                                 |
| `shininess` |                                                     光滑度，指定高光部分的亮度，默认值为 30                                                     |
| `specular`  | 材料高光部分的颜色，默认值为 0x111111 深灰色，如果把它的颜色设置为跟 color 颜色一样的话，会得到类似金属一样的材质，设置成灰色，则看起来更像塑料 |
